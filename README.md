## Dauroh Bot Telegram

### Prerequisite
- python
- telebot
- requests

### Getting started

`pip install pyTelegramBotAPI requests`

### How it works

Terdapat kode untuk pertemuan2 yaitu:

1. Cara mengonsumsi API dengan library `requests`

`python request.py`

2. Cara membuat simple bot menggunakan telebot

`python app.py`
